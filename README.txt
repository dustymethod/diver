Game project done for a class.

Instructions:

	1) Download Processing:
	https://processing.org/
    
	2) Ensure all files are in a directory named "diver".
    
	3) Open any .pde file.
    
	4) Click the "Run" button in the Processing IDE.
	
Controls:
    wasd to move
    space to jump
    mouse button to aim / fire
    hold down longer to throw farther
